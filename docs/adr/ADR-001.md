# ADR-001: Use MCP Runtime Policy Gateway As Product Definition

- Status: Accepted
- Date: 2026-07-01

## Context

Several directions (Android fleet, public-data MCP, scanner-only report) were evaluated. Only inline runtime enforcement is MCP-native and differentiated.

## Decision

The product is an inline MCP policy proxy that controls target tool exposure/invocation and records audit/evidence. Not a scanner, not a vertical app.

## Consequences

All scope and security claims are bound to PROJECT_DIRECTION.md; forbidden claims are CI-checked.
