# ADR-003: Default Deny Policy

- Status: Accepted
- Date: 2026-07-01

## Context

Unknown/hidden target tools must never be exposed or callable by accident.

## Decision

Policy defaults to deny; exposure and calls require explicit allow/alias/approval rules. Default-deny is the real protection surface.

## Consequences

Safety does not depend on name heuristics; it depends on default-deny plus human-curated allowlist.
