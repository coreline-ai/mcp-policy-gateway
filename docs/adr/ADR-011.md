# ADR-011: Use Config Single Principal For Stdio MVP

- Status: Accepted
- Date: 2026-07-01

## Context

stdio provides no upstream caller identity, yet approval/audit schema references tenant/client/actor.

## Decision

In the stdio MVP, tenant/client/actor are a single configured principal. Per-actor enforcement and team audit require an authenticated transport (later).

## Consequences

We do not overclaim per-user enforcement; strong team claims are scoped to managed deployments (see also deployment rule).
