# ADR-015: Fail Closed For Unsupported Reverse Capabilities

- Status: Accepted
- Date: 2026-07-01

## Context

As a client to the target, the gateway may receive server-to-client requests (sampling/elicitation/roots/progress) it does not proxy in MVP.

## Decision

MVP does not advertise these capabilities; unsupported reverse requests fail closed with an audit event. Approval UX uses the gateway_request_approval fallback, not client elicitation.

## Consequences

No hang/MITM ambiguity; reverse-capability proxying is a separate future design.
