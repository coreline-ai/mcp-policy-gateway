# ADR-009: Prohibit Circumvention Targets

- Status: Accepted
- Date: 2026-07-01

## Context

Browser automation / unofficial APIs / paywall / rate-limit / anti-bot / auth bypass targets would make the product an abuse tool.

## Decision

Such targets are unsupported in registration, docs, demos, and sample targets. Only official APIs / explicit permission / owned licenses.

## Consequences

A docs/fixture grep test enforces the absence of circumvention material.
