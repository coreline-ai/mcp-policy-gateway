# ADR-010: Apply Output Policy To Allowed Tool Results

- Status: Accepted
- Date: 2026-07-01

## Context

An allow decision permits the call, not unconditional return of target output.

## Decision

Allowed results pass output redaction/egress/resource-link allowlist before returning; embedded resources default-block. This is best-effort, never a complete DLP guarantee.

## Consequences

Reduces leakage via allowed read tools; tests cover only deterministic cases.
