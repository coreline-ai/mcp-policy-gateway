# ADR-002: Start With Tools-Only Mediation

- Status: Accepted
- Date: 2026-07-01

## Context

resources/prompts carry larger leakage and prompt-injection surface and broader implementation scope.

## Decision

MVP mediates only tools/list and tools/call. resources/prompts are later scope.

## Consequences

Simpler first cut; reverse capabilities (sampling/elicitation/roots) are fail-closed (see ADR-015).
