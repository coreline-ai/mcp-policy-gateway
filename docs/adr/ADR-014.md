# ADR-014: Persist Policy Versions For Audit Reproduction

- Status: Accepted
- Date: 2026-07-01

## Context

Reproducing a decision from an audit event requires the exact policy content in force.

## Decision

policyVersion = HMAC over canonical policy content; the normalized content is stored in mcp_policies keyed by version.

## Consequences

Audit events reference a recoverable policy; already implemented in Phase Entry 1's policy-store.
