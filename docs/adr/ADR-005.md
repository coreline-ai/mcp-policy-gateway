# ADR-005: Store Redacted Audit Metadata By Default

- Status: Accepted
- Date: 2026-07-01

## Context

An audit log that stores raw request/result becomes a sensitive-data honeypot.

## Decision

Default storage is hash + redacted metadata + decision reason. Raw evidence is opt-in, encrypted, short-TTL, separately authorized.

## Consequences

Lower blast radius; read API also applies read-side redaction and RBAC.
