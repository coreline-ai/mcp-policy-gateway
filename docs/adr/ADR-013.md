# ADR-013: Use RFC 8785 Canonical Effective Arguments For Approval Hash

- Status: Accepted
- Date: 2026-07-01

## Context

Loose argument canonicalization breaks replay protection (semantic-equal-but-different-bytes, or collisions).

## Decision

argumentsHash = tenant-scoped HMAC-SHA-256 over RFC 8785 (JCS) canonicalization of the POST-REWRITE effective arguments actually sent to the target. null != missing; forced alias args are included.

## Consequences

Deterministic, testable via JCS vectors; binds the value that actually executes, not raw user input.
