# ADR-006: Use Stdio Target Adapter For MVP

- Status: Accepted
- Date: 2026-07-01

## Context

Remote HTTP targets add SSRF/egress surface and transport complexity.

## Decision

MVP supports stdio target adapters only; HTTP is later, gated behind egress allowlist + private-IP/DNS-rebinding defenses.

## Consequences

Smaller attack surface first; HTTP work is explicitly deferred.
