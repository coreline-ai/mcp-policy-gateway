# ADR-008: Treat Tool List Pagination And List Changed As Catalog Requirements

- Status: Accepted
- Date: 2026-07-01

## Context

Partial observation could hide dangerous tools; silent target changes could bypass policy.

## Decision

Catalog must collect all tools/list pages (nextCursor) before a snapshot is complete, and handle notifications/tools/list_changed. Incomplete snapshots fail closed (calls default-deny).

## Consequences

No hidden-tool window during discovery; changed tools require re-review.
