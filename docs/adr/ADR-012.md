# ADR-012: Treat Target Registration As Privileged Execution Boundary

- Status: Accepted
- Date: 2026-07-01

## Context

Registering a stdio target spawns a process on the gateway host; that is arbitrary code execution.

## Decision

Registry writes are privileged config/operator actions, not runtime user input. Executables are allowlisted; secrets are refs, injected via secret store; all registration is audited.

## Consequences

Registration authz is a first-class security control; HTTP targets add SSRF gates (ADR-006).
