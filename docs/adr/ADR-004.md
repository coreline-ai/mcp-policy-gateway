# ADR-004: Approval Bound To Exact Arguments Hash

- Status: Accepted
- Date: 2026-07-01

## Context

Approved mutations must not be replayed with different arguments or after policy/schema drift.

## Decision

Approvals bind clientId/userId/target/tool + argumentsHash + policyVersion + observationId + schemaHash + rewriteHash, with short TTL and atomic one-time consume.

## Consequences

Replay, race, and stale-schema reuse are blocked. Requires the canonical hash of ADR-013.
