# ADR-007: Require Filtered Target Alias In Public MVP

- Status: Accepted
- Date: 2026-07-01

## Context

A lone gateway.call_tool router looks like a generic proxy, not a policy gateway.

## Decision

The public MVP exposes at least one filtered target alias as a real upstream MCP tool, in addition to the router.

## Consequences

The product's MCP-native differentiation is demonstrable in the demo.
